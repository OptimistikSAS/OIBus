安装要求

让我们在不到2分钟的时间内启动并运行 OIBus。

系统要求​

为了成功安装 OIBus，您的环境应满足以下最低规格：

  • RAM: 2GB。虽然根据您的具体配置，实际需要的 RAM 可能有所不同，但至少需要 2GB。
  • 磁盘空间: 200 MB。您将需要至少 200 MB 的磁盘空间来安装应用程序。然而，建议有额外的空间（根据您的数据流，可能需要几 GB）以储存在上游网络故障时收集的数据。
  • 管理权限：您必须在您的环境中拥有管理权利才能安装 OIBus 并将其注册为服务。这适用于 Windows 和 Linux 平台。
  • 现代网络浏览器：您需要一个现代网络浏览器来访问 OIBus 配置界面。浏览器可 位于与 OIBus 相同的计算机上，或者位于不同的计算机上，只要您配置了 必要的访问设置。注意：不支持 Internet Explorer。
