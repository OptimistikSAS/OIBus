南连接器

了解如何使用和设置南方连接器。

📄️ 概念

南向连接器负责从特定数据源获取数据并将该数据转发到北向缓存。

📄️ 文件夹扫描器

“文件夹扫描器”南向连接器通过定期扫描指定的输入文件夹以查找新文件来运作。它使用其项目中定义的正则表达式（Regexp）来确定要监控哪些文件。当检测到与定义的标准匹配的新文件时，它会被传输到北向缓存。

📄️ OPCUA

OPCUA，即OPC统一架构，是一个旨在读写模式下访问数据的协议。数据在类似树的地址空间中组织，并使用被称为节点ID的唯一标识符引用。OPCUA是基于TCP/IP的现代标准，已取代了早期的OPC HDA/DA技术（参见OPCHDA连接器）。它通常天然嵌入于工业控制器中。

📄️ 微软 SQL 服务器 (MSSQL)

发送 SQL 查询到 Microsoft SQL 服务器。

📄️ Modbus

Modbus 是用于 PLC 网络的通信协议。起初，它是为串行接口开发的，如 RS232、RS422 和 RS485，并后来扩展以支持 TCP 模式。

📄️ SQLite

发送SQL查询以与SQLite数据库交互。

📄️ MQTT

MQTT（消息队列遥测传输）是一种为实时消息和数据交换而设计的协议，它使用发布/订阅模型。在 MQTT 中，数据组织在主题中，客户端可以发布或订阅这些主题来交换信息。

📄️ PostgreSQL

发送SQL查询以与PostgreSQL数据库交互。

📄️ ADS - TwinCAT

自动化设备规范（ADS）协议作为集成到TwinCAT系统中的传输层，由Beckhoff设计并开发。

📄️ MySQL / MariaDB

发送 SQL 查询以与 MySQL / MariaDB 数据库进行交互。

📄️ OPCHDA

OPCDA和OPCHDA是由OPC基金会开发的工业界使用的通信协议。这项技术已被OPCUA取代，但在工业界仍广泛使用。要在OIBus中使用OPCUA，请参阅OPCUA连接器文档。

📄️ Oracle

发送SQL查询以与Oracle数据库交互。

📄️ SLIM (Rest API)

Retrieve data from the Agilent SLIMS

📄️ OIAnalytics (Rest API)

Retrieve data from the Optimistik OIAnalytics application.

📄️ ODBC

向本地 ODBC 驱动程序或可以在服务器同一机器上管理 ODBC 查询的远程 OIBus Agent 发送 SQL 查询。

📄️ OLEDB

Send SQL queries to a remote OIBus Agent that can manage OLEDB queries on the same machine as the server.
