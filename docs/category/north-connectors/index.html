North connectors

Learn how to use and set up north connectors.

📄️ Concept

A North connector is employed to transmit data to a designated target application, extracting the data from its cache.

📄️ OIAnalytics®

Send files and values to OIAnalytics®.

📄️ File Writer

Write files and data to the output folder.

📄️ SFTP

Upload files and data to an SFTP server.

📄️ Amazon S3™

Store files in Amazon S3™ (Simple Storage Service)

📄️ Azure Blob Storage™

Store files in Microsoft Azure Blob Storage™.

📄️ Console

Display filenames or values in Console (used for debug).
