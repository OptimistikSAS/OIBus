# Microsoft SQL Server (MSSQL)
Send SQL queries to Microsoft SQL Server.

## Specific settings
- **Host**: the address of the SQL server
- **Port**: MSSQL server port (default is 1433)
- **Database**: the name of the database to connect to
- **Username**: the username used for authentication
- **Password**: the password used for authentication
- **Domain**: this field is useful when the user wishes to connect to an Active Directory domain
- **Use encryption**: Encrypt the data between the database and OIBus (can overload the server)
- **Trust server certificate**: Accept server certificates even if outdated or selfsigned
- **Connection timeout**: Timeout on connection
- **Request timeout**: Timeout for each query

:::tip Database access
Using a read only user to connect to the database is strongly advised.
:::
