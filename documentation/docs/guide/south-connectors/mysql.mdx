# MySQL® / MariaDB™

Connect to and query MySQL® or MariaDB™ databases with OIBus.

:::info Compatibility Notes

- Supports MySQL 5.7+
- Supports MariaDB 10.2+

:::

## Specific Settings

| Setting            | Description                                                            | Example Value  |
| ------------------ | ---------------------------------------------------------------------- | -------------- |
| Host               | IP address or hostname of the MySQL/MariaDB server.                    | `192.168.1.10` |
| Port               | Server port number. Default: `3306`.                                   | `3306`         |
| Connection timeout | Maximum time in milliseconds to establish connection. Default: `5000`. | `5000`         |
| Database           | Name of the database to connect to.                                    | `production`   |
| Username           | Authentication username.                                               | `db_user`      |
| Password           | Authentication password.                                               | `••••••••`     |

:::caution User Permissions

Always use a read-only database user

:::

:::note MySQL / MariaDB datetime field types
The following **Type** values are available for datetime field configuration:

| Type            | Description                                       |
| --------------- | ------------------------------------------------- |
| String          | String representation parsed with a custom format |
| DateTime        | MySQL `DATETIME` or `TIMESTAMP` column            |
| ISO String      | ISO 8601 string (e.g. `2024-01-15T10:30:00.000Z`) |
| UNIX epoch (s)  | Unix timestamp in seconds                         |
| UNIX epoch (ms) | Unix timestamp in milliseconds                    |

:::

:::note MySQL item request timeout
Each MySQL item has an additional **Request timeout** field (milliseconds) that overrides the connection-level
default for that specific query.
:::
