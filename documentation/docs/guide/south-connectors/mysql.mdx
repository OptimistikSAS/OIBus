# MySQL® / MariaDB™

Connect to and query MySQL® or MariaDB™ databases with OIBus.

:::info Compatibility Notes

- Supports MySQL 5.7+
- Supports MariaDB 10.2+

:::

## Specific Settings

| Setting            | Description                                  | Default Value | Notes                     |
| ------------------ | -------------------------------------------- | ------------- | ------------------------- |
| Host               | Address of the MySQL/MariaDB server instance | -             | IP address or hostname    |
| Port               | Server port number                           | 3306          | Standard MySQL port       |
| Connection timeout | Maximum time (ms) to establish connection    | 5000          | Recommended: 5000-30000ms |
| Database           | Name of the database to connect to           | -             | Case-sensitive            |
| Username           | Authentication username                      | -             |                           |
| Password           | Authentication password                      | -             |                           |

:::caution User Permissions

Always use a read-only database user

:::
