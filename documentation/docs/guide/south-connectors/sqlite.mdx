# SQLite™

Connect to and query SQLite™ database files with OIBus.

## Specific Settings

| Setting       | Description                       | Example Value                    |
| ------------- | --------------------------------- | -------------------------------- |
| Database path | Full path to SQLite database file | `C:\databases\production.sqlite` |

### Best Practices

- Place database on fast storage (SSD recommended)
- Avoid concurrent writes from multiple processes
- Use WAL mode for better concurrency (PRAGMA journal_mode=WAL)
