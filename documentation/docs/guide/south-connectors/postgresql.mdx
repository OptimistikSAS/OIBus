# PostgreSQL

Connect to and query PostgreSQL databases with OIBus.

:::info Compatibility Notes

- Supports PostgreSQL 9.6 and later

:::

## Specific Settings

| Setting            | Description                                            | Default Value | Notes                      |
| ------------------ | ------------------------------------------------------ | ------------- | -------------------------- |
| Host               | Address of the PostgreSQL server                       | -             | IP address or hostname     |
| Port               | Server port number                                     | 5432          | Standard PostgreSQL port   |
| SSL                | Enable encrypted connection                            | Disabled      | Recommended for production |
| Database           | Name of the database to connect to                     | -             | Case-sensitive             |
| Connection timeout | Maximum time (ms) to establish connection              | 10000         | Recommended: 5000-30000ms  |
| Request timeout    | Maximum execution time (ms) for individual SQL queries | 15000         | Adjust for complex queries |
| Username           | Authentication username                                | -             |                            |
| Password           | Authentication password                                | -             |                            |

:::tip Security Best Practices

- Use a read-only database user

:::

### Performance Considerations

**Query Optimization**:

- Use indexed columns in WHERE clauses
- Limit result sets (avoid `SELECT *`)

**Resource Management**:

- SSL encryption adds minimal overhead
- Increase timeouts for large datasets or complex queries
