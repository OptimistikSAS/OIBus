# OLEDB

Connect to OLEDB data sources through a remote [OIBus Agent](../oibus-agent/oledb.md). Particularly useful for systems like OSIsoft PI System™ and other OLEDB-compatible applications.

:::info Compatibility Notes

- Requires OIBus Agent installation on target machine
- Supports all OLEDB-compatible data sources
- Optimized for OSIsoft PI System™ connections

:::

## Specific Settings

| Setting            | Description                                         | Default Value | Notes                              |
| ------------------ | --------------------------------------------------- | ------------- | ---------------------------------- |
| Remote agent URL   | Agent endpoint (e.g., `http://host:2224`)           | -             | Required for remote connections    |
| Connection timeout | Maximum time (ms) to establish connection           | 15000         | Recommended: 5000-30000ms          |
| Retry interval     | Time (ms) to wait before retrying connection        | 10000         |                                    |
| Request timeout    | Maximum execution time (ms) for individual queries  | 15000         | Adjust for large historian queries |
| Connection string  | OLEDB connection string in provider-specific format | -             | See examples below                 |
