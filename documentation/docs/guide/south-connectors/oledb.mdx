# OLEDB

Connect to OLEDB data sources through a remote [OIBus Agent](../oibus-agent/oledb.md). Particularly useful for systems like OSIsoft PI System™ and other OLEDB-compatible applications.

:::info Compatibility Notes

- Requires OIBus Agent installation on target machine
- Supports all OLEDB-compatible data sources
- Optimized for OSIsoft PI System™ connections

:::

## Specific Settings

| Setting            | Description                                                                                            | Example Value                               |
| ------------------ | ------------------------------------------------------------------------------------------------------ | ------------------------------------------- |
| Remote agent URL   | Agent endpoint URL. Required for all connections.                                                      | `http://192.168.1.100:2224`                 |
| Connection timeout | Maximum time in milliseconds to establish connection. Default: `15000`.                                | `15000`                                     |
| Retry interval     | Time in milliseconds to wait before retrying the connection. Default: `10000`.                         | `10000`                                     |
| Request timeout    | Maximum execution time in milliseconds for individual queries. Default: `15000`.                       | `15000`                                     |
| Connection string  | OLEDB connection string in provider-specific format.                                                   | `Provider=OraOLEDB.Oracle;Data Source=mydb` |
| Password           | Authentication password (stored securely). Added as `Password=<password>` if not in connection string. | `••••••••`                                  |

:::note OLEDB datetime field types
The following **Type** values are available for datetime field configuration:

| Type            | Description                                       |
| --------------- | ------------------------------------------------- |
| String          | String representation parsed with a custom format |
| ISO String      | ISO 8601 string (e.g. `2024-01-15T10:30:00.000Z`) |
| UNIX epoch (s)  | Unix timestamp in seconds                         |
| UNIX epoch (ms) | Unix timestamp in milliseconds                    |

:::
