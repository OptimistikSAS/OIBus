# Use Cases

The use cases in this section show end-to-end OIBus deployments with concrete network diagrams, configuration examples, and tips on access rights, firewall rules, and connector tuning.

Each use case covers a specific South → North pairing. Before following a use case, make sure you have read the relevant connector reference pages linked at the top of each guide.

| Use case | South connector | North connector |
| -------- | --------------- | --------------- |
| [OPC UA → OIAnalytics](./use-case-opcua) | OPC UA | OIAnalytics |
| [Microsoft SQL Server → Azure Blob Storage](./use-case-mssql) | MSSQL | Azure Blob Storage |
| [ADS / TwinCAT → OIAnalytics](./use-case-ads) | ADS | OIAnalytics |
| [Modbus → File Writer](./use-case-modbus) | Modbus | File Writer |
| [Aspen IP21 → Amazon S3](./use-case-ip21) | ODBC (via OIBus Agent) | Amazon S3 |
| [OSIsoft PI → OIAnalytics](./use-case-pi) | OSIsoft PI (via OIBus Agent) | OIAnalytics |
| [Advanced multi-network setup](./use-case-advanced) | Multiple | Multiple |

:::tip New to OIBus?
Start with the [OPC UA → OIAnalytics](./use-case-opcua) use case — it covers the most common deployment pattern and introduces the key concepts used across all other use cases.
:::
