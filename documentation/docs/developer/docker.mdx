# Running OIBus with Docker

Deploy OIBus in containerized environments using our official Docker configuration for consistent, isolated installations.

## Official Docker Image

The Docker configuration files are available in the [OIBus GitHub repository](https://github.com/OptimistikSAS/OIBus/tree/main/build/docker/Dockerfile), providing a ready-to-use containerization solution.

### Build Arguments

| Argument  | Description         | Default Value | Example Values     |
| --------- | ------------------- | ------------- | ------------------ |
| `arch`    | System architecture | `x64`         | `x64`, `arm64`     |
| `version` | OIBus version       | `v3.7.0`      | `v3.7.0`, `v3.6.8` |

## Docker Commands

Execute these commands from the directory containing the Dockerfile (`build/docker/`).

### Build the Image

```bash
# Basic build (x64, latest version)
docker build -t oibus .
```

```bash
# Build for specific architecture
docker build -t oibus --build-arg arch="arm64" .
```

```bash
# Build specific version
docker build -t oibus --build-arg version="v3.7.0" .
```

```bash
# Build specific version for ARM64
docker build -t oibus --build-arg arch="arm64" --build-arg version="v3.7.0" .
```
