# Oracle
发送SQL查询以与Oracle数据库交互。

:::caution 依赖性
此连接器需要本地安装Oracle Instant Client（基础版或轻量级基础版，最低支持版本为18.5.0.0.0）
:::

## 特定设置
在配置SQL连接器以发送SQL查询到Oracle服务器时，您需要提供以下细节：
- **主机**：Oracle服务器的地址。
- **端口**：Oracle服务器端口（默认是1521）。
- **数据库**：要连接的数据库名称。
- **用户名**：用于认证的用户名。
- **密码**：用于认证的密码。
- **连接超时**：建立连接的超时设置。

:::tip 数据库访问
强烈建议在连接数据库时使用只读用户，以确保安全和数据完整性。
:::
