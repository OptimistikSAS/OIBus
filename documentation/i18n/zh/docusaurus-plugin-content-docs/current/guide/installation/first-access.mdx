# 首次访问
OIBus配置界面可以在 `http://localhost:2223` 访问。

:::caution 默认访问
默认情况下，用户名是 **admin**，密码是 **pass**。

我们强烈建议在[用户设置](#用户设置)中更改密码。
:::

## OIBus 页面
### 主页
OIBus主页显示启用的连接器和它们关联的指标的引擎。

<div style={{textAlign: 'center'}}>

  ![主页](./first_access/home-page.png)

</div>

放大镜图标将重定向您到连接器或引擎的展示页面。

### 引擎
引擎页面允许您设置日志记录器、[扫描模式](../engine/scan-modes.md)、[IP过滤器](../engine/ip-filters.md)
和[外部来源](../engine/external-sources.md)。

<div style={{textAlign: 'center'}}>

  ![引擎](./first_access/engine.png)

</div>

重启按钮和关闭按钮只影响引擎，不影响服务。这意味着它会关闭连接器，但不会关闭网络服务，这允许您仍然可以访问OIBus界面。

在安装OIBus时，记得选择一个合适的名字。_引擎名称_ 主要在您使用多个OIBus并将日志发送到远程[loki实例](../engine/engine-settings.md#loki)时很重要。

### 北向
要添加一个北向连接器，只需点击 "+" 按钮。

在列表的右侧，您可以选择激活/停用连接器，访问其展示页面或
编辑表单，创建副本或删除它。

<div style={{textAlign: 'center'}}>

  ![北向](./first_access/north.png)

</div>


### 南向
要添加一个南向连接器，只需点击 "+" 按钮。

在列表的右侧，您可以选择激活/停用连接器，访问其展示页面或
编辑表单，创建副本或删除它。

<div style={{textAlign: 'center'}}>

  ![南向](./first_access/south.png)

</div>

### 历史
要添加一个北向连接器，只需点击 "+" 按钮。然后您可以从零开始或通过导入现有的南向或北向设置来创建历史查询。

在列表的右侧，您可以选择激活/停用连接器，访问其展示页面或
编辑表单，创建副本或删除它。

<div style={{textAlign: 'center'}}>

  ![历史查询](./first_access/history-query.png)

</div>

### 日志
日志页面每10秒自动刷新一次。您可以根据日期、日志级别、
日志类型（由南向、北向、引擎等生成的日志）、范围（负责日志的连接器或历史查询），
或日志消息的内容来过滤日志。

<div style={{textAlign: 'center'}}>

  ![日志](./first_access/logs.png)

</div>

### 关于
有关OIBus进程的信息和链接到文档。

</div>
